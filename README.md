# Чат-бот для абитуриентов НИУ ВШЭ

RAG-чат-бот для абитуриентов НИУ ВШЭ (Москва, бакалавриат, 2026). Система отвечает на вопросы о поступлении на основе официальных PDF-документов и страниц НИУ ВШЭ.

Проект является учебным прототипом: он извлекает текст из документов, строит базу знаний, ищет релевантные фрагменты и передает их GPT-4o-mini для генерации ответа.

## Документация

- [Дизайн-документ](docs/design_doc.md)

## Структура репозитория

- `src/` — модули RAG-пайплайна
- `scripts/` — скрипты запуска экспериментов
- `configs/` — YAML-конфиги экспериментов
- `data/` — данные проекта и PDF-документы
- `results/` — результаты экспериментов
- `docs/` — документация
- `report/` — материалы отчета

## Статус

Учебный прототип. Проект не заявляется как полностью готовое production-решение.
